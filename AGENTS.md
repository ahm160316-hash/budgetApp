# AGENTS.md

## 프로젝트 설명
이 저장소는 CSV 기반 가계부 CLI 앱 프로젝트입니다. 거래 데이터를 CSV 파일로 읽고, 추가하고, 조회하고, 집계하는 Python CLI 앱을 개발합니다.

## 코딩 규칙
- 모든 함수에 타입 힌트를 반드시 작성합니다.
- 함수 하나의 길이는 최대 50줄을 넘기지 않습니다.
- 가능하면 작은 순수 함수로 분리해서 읽기 쉽게 유지합니다.

## TDD 규칙
- 구현 전에 반드시 테스트를 먼저 작성합니다.
- 새 기능은 테스트 실패 상태에서 시작하고, 테스트가 통과할 때까지 구현합니다.

## 품질 규칙
- 순환복잡도는 10 이하로 유지합니다.
- 복잡한 로직은 함수 분리로 먼저 단순화합니다.

## 품질 검수 규칙
- 커밋 전에 반드시 `qa_engineer` 서브에이전트로 품질 검수를 진행합니다.
- 검수에서는 테스트 누락, 타입 힌트 누락, 순환복잡도 초과, CSV 처리 엣지 케이스를 확인합니다.

## 테스트 실행 명령어
- `pytest`
- `radon cc`

## 커밋 규칙
- 하나의 기능이 개발되면 즉시 커밋합니다.
- 원격 저장소가 설정되어 있으면 푸시까지 진행합니다.
